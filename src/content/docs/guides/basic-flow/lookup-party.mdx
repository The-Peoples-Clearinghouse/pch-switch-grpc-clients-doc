## Manejo de errores

Si el Switch no puede resolver la consulta, `grpcResponse.getError()` devuelve un objeto con `code` y `message`. De lo contrario devuelve `undefined`.

```typescript

if (error) {
  console.error(`[${error.getCode()}] ${error.getMessage()}`);
}
```
